# Password Checker App

A simple Python command-line application that helps users check the strength of their passwords and validates them against common security rules.

## Features
- Check password length and complexity
- Validate inclusion of uppercase, lowercase, numbers, and special characters
- Provide feedback if the password is weak or strong
- Menu-driven CLI interface

## Technologies Used
- Python
- Loops and Conditional Statements
- Functions

## How to Run
1. Download or clone the repository
2. Run the Python file using:
   ```bash
   python password_checker.py
